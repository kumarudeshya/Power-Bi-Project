# Power-Bi-Project
🚀 Excited to share my latest Power BI project — Customer Behaviour Analysis Dashboard 📊

 ✅ Payment & shipping preferences
 ✅ Customer ratings and subscription behavior
🔍 Key Highlights:
 📌 5K+ customers analyzed
 📌 994K+ total revenue tracked
 📌 Interactive filters for dynamic analysis
 📌 Clean and user-friendly dashboard design
Built using:
 🛠️ Power BI | Data Visualization | DAX | Data Cleaning | Business Intelligence
This project helped me strengthen my skills in transforming raw data into meaningful business insights and creating interactive visual storytelling dashboards.
